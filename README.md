## <br>프로젝트 개요
* **ardiono를 이용한 스마트 탁자 시계** 기능 구현을 목표로 하였습니다.
* 2023 마이크로프로세서 2학년 수업에서 만든 간단한 토이 프로젝트입니다.
* 블루투스 통신을 통해 기능들을 제어할 수 있게 만들었습니다.
* 센서를 통해 탁자 시계 뿐만 아니라 조명의 기능도 할 수 있게 만들었습니다.
